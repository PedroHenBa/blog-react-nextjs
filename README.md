
## Blog made with

- NextJs
- Styled-Components
- TypeScript
- Jest(tests)
- storybook
- disqus-react
- prismjs

<h1>installation</h1>

```bash
git clone https://github.com/PedroHenBa/blog-react-nextjs.git
```
- Install dependencies

```bash
npm install
```

<h1>Getting start</h1>

you can start the server using:

```bash
npm run dev
```

<h1>Customize</h1>

if you want to use your own api, you can change the url in src/config/index.ts
